팀명인 Pathos는 그리스어로 "열정"이라는 뜻을 가진 단어입니다.<br>
또 Path OS, 즉 OS로의 길을 열정을 가지고 개척해 나갈 것이라는 중의적인 의미를 담고있습니다.
